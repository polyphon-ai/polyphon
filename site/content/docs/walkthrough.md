A full product walkthrough from first launch to a live multi-voice session.

{{< video src="/videos/home/full-walkthrough-with-voice.mp4" poster="/images/video-posters/home/full-walkthrough.webp" track="/videos/home/full-walkthrough-narration.vtt" >}}

The walkthrough covers:

- **Onboarding** — setting up your conductor profile (name, pronouns, background context)
- **Providers** — enabling API and CLI voice providers and configuring a custom Ollama endpoint
- **Compositions** — building a named multi-voice configuration with mixed provider types
- **Sessions** — sending a broadcast message, directing a message to a specific voice with @-mention, and watching voices respond in parallel
- **Continuation** — triggering a second round with the Prompt me nudge banner
