Polyphon v0.3.0-alpha.3 is now available.

## What's New

- Migrated to electron-updater for more reliable auto-update with channel support and dev mode
- Added ZIP maker and latest-mac.yml generation for macOS auto-update delivery
- Added update_channel column to user profile so users can opt in to pre-release channels
- Improved composition defaults and the max-rounds slider UX
- Updated about page and update banner to reflect the new electron-updater flow
- Fixed app version injection into the renderer bundle

[Download the latest release](https://polyphon.ai/#download)
