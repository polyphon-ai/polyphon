Polyphon v0.2.0-alpha.5 is now available.

## What's New

- Added macOS code signing and notarization to release builds
- Added YOLO mode for passing arbitrary CLI arguments through to voice providers
- Lazy-load API SDKs on first send and VoiceManager DB data on first session create for faster startup
- Removed unused claude-code provider
- CI now triggers a site deploy after each release
